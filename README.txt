Features
Interactive Gameplay: The gun moves in real-time with the mouse pointer, allowing precise aiming.
Click to Shoot: Players can shoot by clicking anywhere on the game area.
Random Target Movement: Each time a target is hit, it moves to a new random position within the game area.
Score Tracking: The player's score increases with each successful hit.

Future Improvements
Add sound effects for shooting and hitting targets.
Implement different levels of difficulty with varying target speeds.
Introduce a timer or limited ammo for added challenge.
